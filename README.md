# Dogecoin Wallet

Nodejs Dogecoin wallet based on block_io

## Getting Started

To start the wallet simply run

```
npm run runserver
```


### Prerequisites

Nodejs > v8

### Installing

A step by step series of examples that tell you have to get a development env running

Installing nodejs and NPM

```
git clone
```

```
npm install
```


## Deployment

```
npm run runserver
```
